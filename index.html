COVID-19 in Malaysia

About this project

This project was an individual school assignment completed in October 2021.

The objective of this project was to design and create an effective and interactive visualisation that include geographic idioms to present the information and data of a specific domain of our choice while applying the key principles of Data Visualisation.

Project outcomes

I chose the domain of my visualisation to be COVID-19 in Malaysia, with the aim to provide an overview of the pandemic in Malaysia from January 2020 to September 2021.

My visualisation was based on the following 3 datasets:

  • cases_state.csv
  • deaths_state.csv
  • population.csv

You can find my visualisation here , which was created using Vega-Lite.
